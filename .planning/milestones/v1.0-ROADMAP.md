## Milestone Summary

**Key Decisions:**
- **Static Env Vars**: Used `$env/static/public` for build-time validation.
- **OAuth Only**: Prioritized Google login for secure admin access.
- **Premium Aesthetics**: Followed "New Scientist" style using high-contrast typography.
- **Client-side Search**: Implemented reactive filtering for MVP discovery.

**Issues Resolved:**
- Svelte 5 type casting for Appwrite documents.
- Appwrite Cloud connectivity in SSR context.
- Dynamic image resolution from Appwrite Storage IDs.

**Issues Deferred:**
- Server-side global search (Phase 6).
- Multilingual support (Phase 5).
- Fine-grained Svelte 5 transitions (Phase 7).

**Technical Debt Incurred:**
- Manual type casting in some public routes.
- Basic search filtering (non-database indexed).
